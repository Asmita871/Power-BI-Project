# ☕ Coffee Shop Sales Analysis Dashboard (Power BI)

## 📊 Project Overview

This project presents an interactive **Coffee Shop Sales Dashboard** built using **Power BI**.
The dashboard analyzes coffee sales transactions to identify **sales trends, popular coffee types, peak hours, and customer purchasing patterns**.

The goal of this project is to transform raw transaction data into meaningful business insights using **data visualization and analytics**.

---

## 🗂 Dataset Description

The dataset contains coffee shop transaction records with the following fields:

* **hour_of_day** – Hour when the transaction occurred
* **cash_type** – Payment method (Card/Cash)
* **money** – Transaction amount
* **coffee_name** – Type of coffee purchased
* **Time_of_Day** – Morning / Afternoon / Night classification
* **Weekday** – Day of the week
* **Month_name** – Month of transaction
* **Date & Time** – Exact timestamp of purchase

---

## 📈 Dashboard Features

The Power BI dashboard includes several interactive visualizations:

* **KPI Cards**

  * Total Revenue
  * Total Orders
  * Average Order Value
  * Number of Coffee Types

* **Sales Analysis**

  * Revenue by Coffee Type
  * Most Popular Coffee
  * Sales by Weekday
  * Hourly Sales Trend

* **Customer Behavior Insights**

  * Sales by Time of Day
  * Payment Method Distribution

* **Interactive Filters**

  * Month
  * Weekday
  * Coffee Type
    * Time of Day
